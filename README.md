# clonestagram
# 개요
- clonestagram 은 소셜 네트워크 서비스인 '인스타그램' 의 디자인과 주요기능을 React 라이브러리를 사용하여 구현한 프로젝트입니다.
# 사용 기술
### 언어
- Javascript(es7), HTML, CSS, SCSS
### 서버
- NodeJS
### 데이터베이스
- Firebase Realtime Database,
- Firebase Storage
### 라이브러리
- React
- Redux
### IDE
- Visual Studio Code
